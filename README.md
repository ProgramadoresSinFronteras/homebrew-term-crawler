# homebrew-term-crawler

Homebrew tap for [term-crawler](https://github.com/ProgramadoresSinFronteras/term-crawler) — a subversive idle dungeon crawler for terminal workers.

## Install

```
brew install ProgramadoresSinFronteras/term-crawler/term-crawler
```

## Upgrade

```
brew upgrade ProgramadoresSinFronteras/term-crawler/term-crawler
```

This tap is auto-maintained by the source repo's release workflow — `Formula/term-crawler.rb` is rewritten on every `v*` tag.
